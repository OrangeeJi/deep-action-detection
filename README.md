# Multiple Stream Action Recognition

### Prerequisites

* opencv 2.4 with ffmpeg (for gpu_flow)
* cuda + cudnn
* openpose (and thus caffe (all dependencies) + opencv, can be the previous one)
* tensorflow
* pillow
* numpy
* csv
